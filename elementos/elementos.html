Barra de navegación

Es uno de los elementos mas comunes que podemos encontrar en una página web ya que nos brinda una barra comúnmente en la parte superior del sitio el cual nos sirve para poder navegar entre los diferentes apartados que posee la pagina.

Menús Desplegables

Este elemento añade una característica extra al elemento anterior ya que a alguno de los recuadros de la barra de navegación se le agrega otro tipo de sub menú de manera desplegable.

Migas de pan

Este es un elemento muy útil para sitios web, ya que nos permite saber la ubicación de la página en la cual nos encontramos, algo similar como un directorio de carpetas

Paginación

Nos permite tener en botones la cantidad de páginas que dispone el sitio para así poder movernos entre ellas de manera mas fácil e intuitiva.

Botones

Este elemento nos permite agregarle un diseño mas elegante y suave a nuestros botones.

Glyphicons

El elemento de glyphicons es muy atractivo ya que le agrega a los botones una característica que es un icono al botón para hacerlo mas intuitivo en relación a la acción del botón, bootstrap cuenta con aproximadamente 180 glyphicons.

Uso de Vanilla JS

Se quito la dependencia forzada de JQUERY, la cual se usaba para las animaciones en menús desplegables, lo cual no fue del agrado de mucha gente, actualmente se pueden usar marcos de JS mas modernos como Vue.js o react

Compatibilidad IE 10 y 11

Se elimino la compatibilidad con el navegador internet Explorer en las versiones 10 y 11, esto aumento las opciones de diseño de CSS, carga mas rápida en el uso de JS y mejoras en el uso de las API debido a que estas ya no tienen que ser optimizadas para este navegador

Nuevo acordeon

El nuevo acordeón incluye Bootstrap Iconos que indican el estado y la capacidad de hacer clic. También puedes usarlo para eliminar el predeterminado, algunos bordes y algunas esquinas redondeadas para renderizar acordeones de borde a borde con su contenedor principal

Etiquetas flotantes

También se ha agregado un nuevo soporte de etiqueta flotante en bootstrap 5.

Marcador de posición

Se ha agregado un nuevo componente de marcadores de posición que te permite crear una interfaz de usuario mientras tu aplicación aún se está cargando.

soporte RTL

Bootstrap 5 ahora incluye soporte RTL. La compatibilidad con RTL (de derecha a izquierda ) significa que puede crear sitios web en idiomas como árabe, hebreo, pashto, persa, urdu y sindhi. Los escritos en estos idiomas comienzan desde la derecha de la página y continúan hacia la izquierda

Formularios actualizados

La apariencia de los formularios se ha actualizado y además se ha agregado el botón Switch.

Paleta de colores

Bootstrap 5 incluye 21 variables de colores en su codigo CSS.
